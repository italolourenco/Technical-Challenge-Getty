# Technical Challenge

## First Steps 

### Clone Repository

```bash
git clone https://github.com/italolourenco/Technical-Challenge-Getty.git
cd Technical-Challenge-Getty

```

### Dependencies Installations

```bash
-> npm install 

```

### Run Server

```bash
-> json-server server.json -p 3001 -w -d 500
```

### Run App

```bash
-> npm start
```
